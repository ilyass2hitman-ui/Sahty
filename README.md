## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/ilyass2hitman-ui/Sahty.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Sahty
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
